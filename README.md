# router
需求: 将linux系统配置成一台IPV4路由器,具备RIP,OSPF(单区域)和路由重分配功能.搭建至少6个路由器节点的实验测试网络.
关键词:
IPV4路由器
RIP (Routing Information Protocol) 每隔30秒会与相邻的路由器交换子消息，以动态的创建路由表。
OSPF (Open Shortest Path First) 基于IP协议, 开放式最短路径优先
路由重分配
